# DSA Interview Preparation

This directory contains structured notes, problem-solving records, mistake analysis,
pattern summaries, and review material for coding interview preparation.

## Structure

- `roadmap.md` — topic progression and preparation status
- `patterns/` — reusable algorithmic patterns
- `problems/` — individual problem records
- `mistakes/` — recurring reasoning and implementation mistakes
- `reviews/` — scheduled review material and retrospective notes

## Principles

1. Attempt problems before reading solutions.
2. Record reasoning, not just final code.
3. Explain correctness and complexity.
4. Track recurring mistakes.
5. Revisit weak patterns using spaced review.
6. Never copy paid or copyrighted solution material into this repository.
